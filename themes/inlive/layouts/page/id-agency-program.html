Program Kerja Sama Agensi Indonesia adalah program kerja sama antara inLive dan para agensi perangkat lunak untuk membantu bisnis atau organisasi di Indonesia dalam menyelesaikan tantangan dan kendala dalam real-time communication dan live streaming seperti pusat bantuan online, kelas online, konsultasi online, dan lainnya. Saat ini sebagian besar masalah tersebut ditangani dengan aplikasi pihak ketiga seperti WhatsApp, Zoom, ataupun Google Meet.Â 

Aplikasi tersebut berjalan cukup baik dari sisi fungsi, namun pada saat penggunaan makin bertambah, maka kebutuhan melakukan otomatisasi layanan akan sangat diperlukan. Sehingga diperlukan integrasi sistem dengan layanan real-time communication dan live streaming tersebut yang memungkinkan untuk melakukan monitoring dan otomasi yang lebih baik.

inLive adalah layanan real-time communication dan live streaming berbasis Cloud API (Application Programming Interface) yang memungkinkan sebuah aplikasi memiliki fitur seperti WhatsApp, Zoom, atau Google Meet yang terintegrasi langsung dengan aplikasi tersebut. Hal ini memungkinkan untuk melakukan otomasi dan monitoring yang lebih baik dalam penggunaan fitur tersebut tanpa harus berpindah ke aplikasi pihak ketiga.

Layanan inLive bisa dipelajari lebih lanjut di halaman ini.

Bentuk kerja sama _

Saling menguntungkan dalam pengembangan bisnis Anda.

Dalam program ini, kerja sama antara inLive dan agensi perangkat lunak adalah inLive sebagai penyedia layanan akan berbagi hasil pendapatan dengan para agensi untuk setiap aplikasi yang diimplementasikan oleh para agensi untuk client-nya. Besaran pembagian hasil akan disepakati bersama dalam rentang 10-20% dari biaya yang dibebankan inLive ke pengguna Cloud API inLive.

Benefit untuk setiap agensi yang menjadi bagian dari program ini

  • Sistem bagi hasil 10-20% dari setiap biaya yang dibebankan ke aplikasi client

  • Training pengembangan aplikasi berbasis WebRTC dan penerapan Cloud API inLive

  • Dukungan teknis gratis dan update berkala dari inLive

  • Akses ke versi beta dari semua produk inLive lebih awal

Ayo bergabung dalam program kerja sama ini

Form pendaftaran

Setiap agensi yang bergabung harus memiliki kualifikasi sebagai berikut:

  • Berbadan hukum PT (Perseroan Terbatas)
  • Memiliki portofolio aplikasi/client profesional lebih dari 5 aplikasi/client berbasis web (JavaScript) atau mobile (iOS/Android)
  • Memiliki pengalaman dalam menggunakan Cloud API

Untuk setiap agensi yang memenuhi kualifikasi di atas dan ingin bergabung dengan program ini bisa mendaftarkan perusahaan atau agensi terkait dengan mengisi form pendaftaran.
